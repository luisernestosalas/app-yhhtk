# CRM con Clientes y Notas

Una aplicación web simple y eficiente para la gestión de clientes y sus notas asociadas, construida con tecnologías web estándar.

## 🚀 Demo en Vivo

Puedes acceder a la aplicación en: [https://app-yhhtk-c4fefu0t1-luisernestosalas-2775s-projects.vercel.app](https://app-yhhtk-c4fefu0t1-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- **Gestión de Clientes**: Agregar, editar y eliminar información de clientes
- **Sistema de Notas**: Crear y gestionar notas asociadas a cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador del usuario
- **Responsive**: Compatible con dispositivos móviles y de escritorio
